
  

# Front End
Start the front-end

1. Clone Repository

`git clone https://github.com/dormammuuuuu/SADI.git`

2. Change directory to front-end

`cd sadi-react`

3. Download packages

`npm install`

4. Run server

`npm start`

  

# Back End
Start the back-end
1. Change directory to back-end

`cd sadi-flask`

2. Install Packages

`pip install -r requirements.txt`

3. Run Server

`python app.py`
